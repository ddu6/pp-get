# PKU Playback Getter
- Require [`nodejs`](https://nodejs.org/).

## Install
```
git clone https://github.com/ddu6/pp-get.git
```
```
cd pp-get
```
```
npm install
```
```
npm run init
```
Modify `passwords.csv`.

## Use
```
npm run collect
```
Check and modify `classes.csv`.
```
npm run download
```
Videos listed in `classes.csv` will be downloaded to `archive`.
