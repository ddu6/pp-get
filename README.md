# PKU Playbacks Getter
- Require [nodejs](https://nodejs.org/).

## Install
```
git clone https://github.com/ddu6/pp-get.git
```
```
cd pp-get
```
```
npm install
```
```
npm run init
```
Fill in `passwords.csv`.

## Use
```
npm run collect
```
Check `lessons.csv`.
```
npm run download
```
Videos listed in `lessons.csv` will be downloaded to `archive`.
